end

Properly closes functions and loops, not the same as endheader.

print [arg1] [arg2] [...]

This command translates to "System.out.println()" in Java. The arguments are concatenated with a space added after each one.
Ailiases: prt, p

pvar [variable]

Fstrings and the like are currently unavailable in whatever I'll call this sub-language. You must use pvar to concatenate.
Ailiases: pvar, pv

var [variableName] [value]

This command creates a variable without requiring the type. Since Java is type specific of course, the type will still be determined automatically.
Ailiases: let

for

This command simply declares a for loop. There are sub commands for using it.

from [start] to [end]

This assumes a starting variable of i. An embedded loop should increment the character to j, then k, then so on. It will count backward if necessary. To be inclusive, add "inc" or "inclusive" to the end of the statement

from [start] forever

This increments the counter and repeats the loop forever.

java [args1] [...]

This is your failsafe for when you can't use functions or commands. It might be convenient for multi-line code, though. Ailiases: j

read [types] [var name]

This creates one scanner object in your program. Then, if one doesn't already exists, it creates a variable with the given name and type to put the value of the scanner call into. Ailiases: None

ask [args1] ...

This prints like you'd expect from any print command, but with the benefit of creating a newline and a > character, allowing you to fancify your input!

rand [variableName] [min] [max]

Generates a random number from the desired minimum to maximum.

out [args1] ...

Returns a value from a function

func [returnType] [methodName] [arg1Type] [arg1Name] [arg2Type] [arg2Name] ...

Returns a value from a function
